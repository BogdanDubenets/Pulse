# Навичка: Обробка новин Pulse

Ця навичка визначає, як інтелектуально обробляти великі потоки новин для створення преміального контенту.

## Коли використовувати
- При розробці логіки збору новин через Telethon.
- При налаштуванні AI-промптів для резюмування.
- При реалізації алгоритмів дедуплікації на основі векторних embeddings.

## Ключові принципи
1. **Дедуплікація (Clusterization)**:
   - Використовувати Cosine Similarity (поріг > 0.85) для об'єднання постів в одну "Історію" (Story).
   - Пріоритет - виявлення першоджерела.
2. **Резюмування (Summarization)**:
   - Формат: "Що сталося" + "Чому це важливо" + "Джерела".
   - Уникати клікбейту, зберігати нейтральний інформаційний стиль.
3. **Confidence Score**:
   - Розраховувати довіру на основі кількості джерел та їхньої репутації (credibility).
