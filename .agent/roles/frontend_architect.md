# Роль: Frontend Architect (Pulse)

## Обов'язки
- Проектування та розробка Telegram Mini App (React + TypeScript + Vite).
- Дизайн UI/UX веб-інтерфейсів з дотриманням дизайн-системи Pulse.
- Масштабування Mini App у повноцінний веб-ресурс (лендінги, дашборди, портал).
- Інтеграція з Telegram WebApp SDK та бекенд API (FastAPI).
- Забезпечення адаптивності та мобільної оптимізації.

## Навички (Skills)
- `pulse-design-system` — палітра, типографіка, компоненти, брендинг
- `miniapp-development` — архітектура, Telegram SDK, конвенції, деплой

## Технологічний стек
- React 18+ / TypeScript 5+
- Vite 6+ (build + dev server)
- Tailwind CSS 4+ (з CSS-змінними)
- Lucide React (іконки)
- Framer Motion (анімації, в перспективі)

## Зона відповідальності
- `pulse-miniapp/` — весь фронтенд
- Кольорова палітра та стилізація
- Favicon, лого, брендинг-ассети
- UX-патерни: навігація, перемикачі, модалки
- Деплой на Vercel/Netlify
